# Right-Player
一款轻量级音乐播放器，拥有简洁美观的UI设计  
调用Win10未公开API(SetWindowCompositionAttribute)实现磨砂效果  
使用共享内存实现单例模式  
（记得先安装解码器[LAV Filters]）
